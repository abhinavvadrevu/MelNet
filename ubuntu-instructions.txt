0. Clone the git repo over
`ssh-keygen -t rsa -b 4096 -C "abhinavvadrevu1@gmail.com"`
Press Enter to accept default locations

1. Copy the data over as a zip file

`scp -i ~/.ssh/chronicle-test-keys.pem ./datasets/KSS.zip ubuntu@ec2-54-157-191-54.compute-1.amazonaws.com:/home/ubuntu/workspace/MelNet/datasets/`

2. Install ffmpeg
`sudo apt update`
`sudo apt install ffmpeg`
Check ffmpeg version with ffmpeg -version

3.  Run the thing

`python trainer.py -c ./config/kss.yaml -n kss_t1_b1 -t 1 -b 1 -s TTS`
`python trainer.py -c ./config/kss.yaml -n kss_t2_b4 -t 2 -b 4 -s TTS`

4. Copy the file back to laptop before shutting the instance down

`scp -i ~/.ssh/chronicle-test-keys.pem ubuntu@ec2-54-157-191-54.compute-1.amazonaws.com:/home/ubuntu/workspace/MelNet/chkpt ~/workspace/MelNet/chkpt`

5. `python inference.py -c ./config/kss.yaml -p ./config/inference.yaml -t [timestep of generated mel spectrogram] -n [name of sample] -i [input sentence for conditional generation]`
